# 开发技术栈
语言：Java
框架：Spring boot 3.2.4,  Spring Security, MybatisPlus, MyBatis
数据库：MySQL 8.0.23
数据库版本管理：flyway
缓存: Redis
测试框架：Junit, Mockito, Spring Test
其它：lombok, swagger, logback, jackson 3.0,

# 开发规范
- 代码风格统一，命名规范
- 文件根据domain进行分类，相同实体和对应的Mapper、Service、Controller放在一起
- controller层主要负责校验、转发请求和组装返回结果，不包含业务逻辑和数据库操作，只负责调用service层
- 方法注释详细，参数返回值说明
- 异常处理统一，返回格式一致
- 测试用例覆盖所有接口和分支
- 代码提交前需保证编译通过、所有单元测试通过
